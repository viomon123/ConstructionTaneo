# Construction Inventory System

A React application built with Vite and Tailwind CSS for managing construction inventory and expenses.

## Features

- **Inventory Management**: Track items with name, quantity, price, and total cost.
- **Daily Changes Tracking**: Log changes in quantity and price with dates.
- **Expense Classification**: Categorize and track expenses monthly.
- **Persistent Storage**: Data is saved in localStorage.

## Getting Started

### Prerequisites

- Node.js (version 18 or higher)
- npm

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Running the Application

Start the development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```

Preview the production build:
```bash
npm run preview
```

## Usage

- Switch between Inventory and Expenses tabs.
- Add, edit, or delete inventory items.
- Add expenses with categories and dates.
- View monthly expense summaries.

## Technologies Used

- React
- Vite
- Tailwind CSS
